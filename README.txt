'Wobble' correction software to correct z-dependent axial abberation in 
STORM/ PALM data.

The matlab code contains full documentation and examples of usage on 
the suppplied test data. Please type 'doc wobbleCalibration' or 
'doc correctWobble' at the MATLAB command prompt to access the documentation
